# Overview

```
The purpose of this lab is to develop a responsive web application using Vue.js 3 and
Bootstrap that allows users to search for events and provides a registration form.
```
# 1. Event Information (Searchable Table)

```
The event data are stored in the events.txt file. Initially, display all events from events.txt in a
table. The table must include four columns:
```
- Event ID
- Event Name
- Category
- Duration (Hours)

```
Create separate text input fields for searching by Event ID, Event Name, and Duration. Use
radio buttons for filtering by category (Technology, Business, Marketing, Finance, All).
```

# 2. Registration Form

```

```
- A username input field

- Radio buttons to select an event category
- A drop-down list for event names, populated dynamically based on the selected
    category
Once the user makes a selection, display a summary message showing:
